可视化Nginx代理神器 Nginx Proxy Manager

发表于 2022-04-27 更新于 2022-06-20

今天一直被开发催着帮忙配置Nginx，一群开发竟然没一个会Nginx这种入门技能的。很是意外，Nginx应该是属于初级后端工程师和中高级前端工程师必备的技能，即便是基于docker容器。原本想做一期Nginx入门培训，后来想想先搜索下有没开源的nginx的管理软件，还真在github上找到了 —— Nginx Proxy Manager。Nginx Proxy Manager就是一款让你能通过网页的一些设置，完成网站的代理配置，无需自己再手动安装Nginx修改配置文件了，大大提高了效率。项目也是开源的，不用担心项目的安全性。有了这玩意，还要什么自行车。
