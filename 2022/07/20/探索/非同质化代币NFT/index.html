非同质化代币NFT

发表于 2022-07-20 分类于 探索

以下内容主要摘录自OKX 交易所，请遵守法律合规，谨慎搭梯浏览。

什么是 NFT？

NFT的全名为Non-FungibleToken，中文意思是非同质化代币，是一种具备唯一性和不可分割性的代币，每一枚代币都是独特的，不可分割和互换的。

NFT VS FT

  • NFT 即非同质化代币，是唯一的、不可拆分的 token，如加密猫、token 化的数字门票等同质化代币。

    加密猫？
    2017 年 11 月 28 日一款基于以太坊网络名为 CryptoKitties 的游戏公开上线，这是一款养猫游戏，每只猫具有不同的基因，具有独一性，且可以使用 ETH 自由交易，该游戏已经推出，便火爆市场，随着玩家的涌入，每只加密猫的价格不断水涨船高，其中以太猫创世 1 号 Genesis 拍出了 246.95 个 ETH 的天价，当时折价人民币 77 万！

  • FT（Fungible Token）即同质化代币是一种能够替换、具有统一性、可接近无穷拆分的代币，比如 BTC、ETH 都是同质化代币。

举个栗子：
每个人所持有的比特币，全都是一模一样、价值相同，交易时也可以被分拆，比特币就像我们现实花钱一样，可以是 0.001BTC；而 NFT 则相反，每一个 NFT 都是独一无二整体、不可相互替代的，而且交易时是整体转移的。

NFT 的优势

唯一性，不可分割性，所有权，跨游戏属性。NFT 其实也是一种 Token，但和 FT（同质化代币）不同的是，每一个 NFT 都有着自己区别于其他 NFT 的独特属性，每一个 NFT 都被打上了一个唯一性的烙印，这个烙印决定了每一个 NFT 的价值都不一样。

NFT的出现解决了虚拟世界里的物权问题，我们在现实世界的房子、车等固定资产可以通过购买发票或者产权证明来佐证我们对于货品是有所有权的。但是虚拟世界的产出物就很难定义产权问题，过去一直没有一套理想的机制来形成保护链条。这个原因也造成了 NFT 概念出现后的爆发式增长，利用区块链技术的公开透明，不可篡改的特性，把虚拟世界产出的音乐、艺术创作、卡牌、等等虚拟物品，形成一套完整的版权机制，用来形成对于虚拟世界物权的所有证明。

ERC721 和 ERC1155

ERC721是第一个 NFT 标准，拉开了 NFT 热潮。ERC1155是一个更灵活的 NFT 标准，允许同一个不可替代令牌的不同版本。详情戳这里ERC721 与 ERC1155 有什么区别。

  • ERC721
    1. 仅限于非同质化代币；
    2. 只支持每次传输一个代币；
    3. 一份智能合约只能发行一种代币。
  • ERC-1155
    1. 允许同质和非同质；
    2. 允许像半同质代币这样的新概念币，例如，同质代币可以“转化”为非同质代币，反之亦然。支持在单个交易中批量传输多个代币 ID。（由于每个以太坊交易大约要 15-30 秒的时间，因此 ERC-721 需要花很长时间来传输代币，而 ERC-1155 可以在一个区块中发送数百个不同的代币。优化的 ERC-1155 顺利通过测试，每秒可传输 150-200 个代币）；
    3. 一个智能合约可发行多种代币。

国内政策法规

NFT在中国的发展路径将遵循不同于海外市场的商业模式，中国企业更多是从版权保护切入，发挥NFT数字产权证明功能，强调无币化NFT的探索。由于中国对于虚拟资产的监管态度一直是比较严厉，政策面比较严谨，在比特币的快速爆发后，国家快速出台了相关政策，2021 年 9 月 24 日，国家发改委等 11 部门发布《关于整治虚拟货币“挖矿”活动的通知》，宣布虚拟货币“挖矿”活动将被正式列为淘汰类产业；同时，央行等 10 部门发布《关于进一步防范和处置虚拟货币交易炒作风险的通知》明确虚拟货币不具有与法定货币等同的法律地位，同时把相关业务定义非法金融活动。虽然现阶段我国对于NFT拍卖还没有严格的法律条文约束，但是NFT与区块链技术的深度结合，随着NFT应用的大规模推广，未来对于NFT的铸造、发行、销售、流转都一定会有监管的介入。

2022 年，4 月 13 日，中国互联网金融协会、中国银行业协会、中国证券业协会联合发布《关于防范 NFT 相关金融风险的倡议》，提出NFT作为一项区块链技术创新应用，在丰富数字经济模式、促进文创产业发展等方面显现出一定的潜在价值，但同时也存在炒作、洗钱、非法金融活动等风险隐患。

目前从国家政策导向及持牌请来说，NFT在中国的交易，目前仍属于未定性阶段，若将 NFT 当做自带金融属性的特殊虚拟商品，二手交易需获取国家级的金融牌照。若仅将 NFT 当做普通虚拟商品，自身不具有金融属性，其二手交易或将被允许。现在来看因为 NFT 投资是有金融波动性的，未来定义成金融属性的虚拟商品可能性较大。同时我国并没有专门针对NFT交易所的牌照和相关管理制度，现市场上存在的NFT交易市场的行为都有不合规的潜在风险。

NFT 平台开展业务所需证照及主管部门

根据相关证照、备案要求及实践来看，NFT 所需部分证照的获取具有一定难度。目前来看，中国法律对于铸造及首次发行 NFT 资产并自行持有以及偶发性熟人之间的交易容忍度相对较高。

国内主流的 NFT

我国的 NFT 铸造主要分两种模式分别是 PGC 和 UGC，区别主要是艺术家发行和个人用户上传。PGC 主要是专业人士和平台进行合作，发现作品 NFT，平台和专业人士共同进行销售分成。国内例如，阿里的蚂蚁链、 腾讯的幻核、以及京东的灵稀等。UGC 模式则是由用户自主创造内容，在平台发布，供需双方在平台自由采购，平台赚取服务费。由于 UGC 在国内政策风险较大，所以模式的头部企业基本都是国外公司。
对比全球市场，中国市场在 NFT 的交易上最大的差异体现在区块链、交易货币、交易涨跌幅以及版权问题。中国的NFT的区块链底层服务还没有公联支持，主要以联盟链为主，其中蚂蚁链和长安链是做的规模较大的。同时中国市场的 NFT 交易货币主要是人民币，而国外市场主要是以几大主流虚拟货币为主。而国内几大主流市场现阶段皆不支持二次交易，主要是以收藏为主，这个政策导向息息相关。对于未来希望从事 NFT 二手交易的 B 端用户还是 C 端用户来说，都不太乐观。
