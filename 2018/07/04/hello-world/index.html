Hello World

发表于 2018-07-04 更新于 2022-06-20

之前都是在印象笔记上记录自己的笔记、心得和踩过的坑，一直想转移到博客上。但由于没有挑到喜欢的开源博客系统，自己又有严重的强迫症不愿意妥协，加之工作也挺忙（这个是借口）一直没有落实。

最近公司要注销，留下了些许人给做集团交接，现已经到了尾声，闲的发慌便来研究。请教了公司的运维大牛H22F，给我推荐了基于Github Pages的Hexo博客发布工具。了解了一下，甚是喜欢，决定使用Hexo来搭建自己的博客。

Hexo是一个简单、快速、强大的基于 Github Pages 的博客发布工具，支持Markdown格式，有众多优秀插件和主题。
其实很多我们经常光顾的博客都是基于Hexo搭建的，比如阿里间件团队博客等，只是自己从来没有关注过页面最下面那行小字“由 Hexo 强力驱动”或者“Powered by Hexo”。
